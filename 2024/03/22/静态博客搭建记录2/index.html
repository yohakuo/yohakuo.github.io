基本操作

问题解决

You have unmerged paths.

  • 查看下暂存区修改后的文件状态。
  • 来到标红的文件进行修改，再次 push **MERGING** 就没有了

git push -u origin --> fatal: unable to access

  • 远程仓库和本地不一致，比如远程有的文件本地没有
  • 查询并取消代理
  • 修改dns
